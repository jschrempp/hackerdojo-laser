On startup:

Turn on three power switches

  • 2 switches on right side near knee height for machine and lazer
  • 1 switch on the chiller. It will briefly emit noise and red lights

Load Work to software

Files accepted:
  • .rd /.rdl : native files for the RD viewer
  • .ai : adobe illustrator. Combines layers of vector and raster
  • .dxf : autocad / autodesk exchange files. Combines layers of vector and raster.
  • .jpeg / .png : raster only. By default transparency not supported. May be possible to use pre-processing to create vector outlines in software tools such as Adobe, CAD, acorn, etc.
  • "Corel" : Not sure
Files NOT accepted:
  • .svg : scalable vector graphics, a possible output from CAD and AI. Can contain raster. No support.
  • .fbx : 3d file interchange format. No support.
  • .collada, .3ds, .blend, .obj, .metal : Other 3d formats. No support
  • files of type video, audio, text, etc. not usually for image processing

Level U and Z

Download file from software to machine

Press frame to test machine

Turn off machine

File Cleanup on Machine

On control panel, press esc until the left side is blank and the right contains a few fields with numbers. Press the "FILE" button. Press the "RIGHT" button. Check to see the files in queue show up. For each file, press left to hover over it. Press enter. Navigate on the machine to Delete. Press enter. Repeat to confirm. Do this before running any new file or any file with changes in it.
